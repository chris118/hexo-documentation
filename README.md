## Getting started

Install dependencies:

``` bash
$ git clone http://code.putao.io/hexo-documentation
$ cd hexo-documentation
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
